# Iris Flower Classification

## Project Description
Machine learning project to classify iris flowers into three species (Setosa, Versicolor, Virginica) 
using their physical measurements. Built with Python and scikit-learn for QSkill internship.

## Dataset
- **Source:** Scikit-learn built-in Iris dataset
- **Samples:** 150 iris flowers
- **Features:** 4 (Sepal Length, Sepal Width, Petal Length, Petal Width)
- **Classes:** 3 species

## Features
- Data exploration and visualization
- Feature scaling with StandardScaler
- Multiple classification models (Naive Bayes, Decision Tree, Random Forest, SVM)
- Model evaluation with accuracy, precision, recall metrics
- Confusion matrix visualization

## Models Trained
1. **Naive Bayes** - 96.67% accuracy
2. **Decision Tree** - 96.67% accuracy
3. **Random Forest** - 98.00% accuracy ⭐
4. **Support Vector Machine (SVM)** - 98.00% accuracy ⭐

## Best Model
**Random Forest Classifier** with 98% accuracy
